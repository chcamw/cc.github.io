【蔬食文化/無肉時代的來臨】

2020年統計台灣吃素人口佔13%，已突破三百多萬人，意味著在台灣每十個人就有一個人吃素，這樣的素食者人口比例僅次於印度、墨西哥。

不只是原生家庭、宗教因素的茹素者，現今有越來越多的人為了環境保育及動物保護的理念，漸漸開始執行「無肉」飲食或「蔬食」飲食。

 

【未來肉大勢】

未來肉、未來漢堡、植物肉、新豬肉，不知道大家有沒有聽過、看過或吃過這種食物呢？

 

現在市面上有非常多這類型的素食食品，而且吃起來越來越“真實”！他們做得非常像一般的肉排、肉球，不只是外觀，甚至連口感和香氣都做得到位。

 

這些“未來肉”的主要成分是使用植物蛋白，像是豌豆蛋白、大豆蛋白、綠豆蛋白等來源製成的，所以蛋白質含量是很充足的，也有許多產品可能會使用「營養強化」的方式，添加一些維生素、礦物質來補足素食者易缺乏的營養素。

 

但為了產生“肉”的“口感”、“氣味”和“外觀”，我們需要考量到它可能使用了大量的食品添加物來製造，鈉含量和油脂（通常使用芥花油、椰子油等）也不見得會比真實的肉來得少。

 

所以素食者偶爾吃這樣的食品增加飲食多元性的確是可行的，而且在製程上的確未來肉比“漢堡肉”更接近環保的理念，大大地減低了碳排放量。

看完了這些小介紹，你是不是也想馬上試試未來肉呢？最終仍要提醒植物肉畢竟是加工食品，營養師不建議大家天天吃這樣的食材，每天飲食還是以“原型食物”為主會更好！

 

-

吃素就一定健康嗎？還在以為吃素就只是吃能吃菜？

 

【蔬食者更要注重的均衡飲食】

我們在嘗試吃素前，其實可以大大的撇除「吃素=吃菜」這樣的迷思，不用害怕很像餐餐都只有青菜可以吃，素食飲食也能夠很多元！

 

*每日蛋白質需要量，以蛋奶素者為例

蔬食女性 平均攝取 1500kcal/day

每日（豆+蛋）共4份

每日乳品類 共1.5份

 

蔬食男性 平均攝取 2000kcal/day

每日（豆+蛋）共6份

每日乳品類 共1.5份

 

蔬食蛋白質來源：一份(7g)蛋白質 = 1顆蛋 = 6顆鵪鶉蛋 = 1杯190ml豆漿 = 半盒嫩豆腐 = 1片豆包 = 1.5塊小方豆乾 = 1/4片大黑豆干 = 20g黃豆、25g黑豆、50g毛豆

-

素食飲食指標：

1. 依據指南擇素食，食物種類多樣化。

2. 全穀雜糧為主食，豆類搭配食更佳。

3. 烹調用油常變化，堅果種子不可少。

4. 深色蔬菜營養高，菇藻紫菜應俱全。

5. 水果正餐同食用，當季在地份量足。

6. 口味清淡保健康，飲食減少油鹽糖。

7. 粗食原味少精製，加工食品慎選食。

8. 健康運動3 0 分，適度日曬2 0 分。

若能夠做到這八大項飲食原則，即使素食者也不用擔心營養不均衡的問題😉
